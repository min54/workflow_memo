## 구현 완료 기능

### 캔버스 기본
- 무한 캔버스 (마우스 휠 줌, 좌클릭 드래그 패닝)
- **좌측/상단 경계 고정**: tx, ty ≤ 0 → 왼쪽·위로 무한 스크롤 불가
- **시작 시 항상 100% 줌**, 원점(0,0) 고정
- **프로젝트 이름 워터마크**: 캔버스 좌상단(0,0) 고정, 120px, 투명도 30%

### 텍스트 블록
- 더블클릭 → 텍스트 블록 생성 (클릭 위치 정확하게 보정)
- 생성 직후 커서 자동 진입
- 빈 블록은 포커스 벗어나면 자동 삭제
- **슬래시 커맨드**:
  - `/섹션` + 스페이스 → 39.6px (1.8em), font-weight 800
  - `/제목` + 스페이스 → 30px (1.364em), font-weight 700
  - `/설명` + 스페이스 → 기본(22px)으로 복귀
- **폰트 크기**: 설명 22px / 제목 30px / 섹션 39.6px
- **줄간격**: 1.4 (모든 텍스트 동일)
- 블록 이동 핸들 (크게 설정됨)

### 파일 업로드
- 업로드 전 파일명 직접 입력 모달
- 이미지: base64 JPEG 변환, 캔버스 + 하단 스트립 배치, border-radius 3px
- PDF, Excel, Word, PPT, TXT 업로드 지원
- 하단 스트립에서 파일 열기/다운로드

### 그룹 선택 (S 툴)
- 러버밴드 드래그로 다중 블록 선택
- 빈 캔버스 드래그 → 그룹 이동
- 핸들 드래그 → 항상 단독 이동 (그룹 자동 해제)
- 툴바 휴지통 or Delete/Backspace → 선택 블록 일괄 삭제
- 삭제 시 하단 스트립 카드도 함께 제거

### 복사 / 붙여넣기
- Ctrl+C → 선택 블록 복사 (텍스트·이미지·파일 모두 지원)
- Ctrl+V → +30px 오프셋으로 붙여넣기
- **프로젝트 간 붙여넣기 가능** (같은 탭 내, 새로고침 전까지 유지)
- 툴바 복사 버튼도 동일 동작

### 실행 취소 (Ctrl+Z)
- 최대 20단계 Undo 스택 (JSON 스냅샷 방식)
- 블록 삭제, 생성 등 주요 동작 복구 가능

### 전체 검색 (Ctrl+F / 사이드바 돋보기)
- 전체 프로젝트 localStorage 대상 검색
- 텍스트 블록 제목+내용, 이미지 캡션, 파일명 검색
- 결과 프로젝트별 그룹핑, 매칭 텍스트 노란 하이라이트
- 결과 클릭 → 해당 프로젝트로 이동 + 블록 위치 점프 + 노란 테두리 강조

### ZIP 다운로드
- 툴바 다운로드 버튼 → `프로젝트명.zip`
- `canvas.html`: 블록들을 wx,wy 좌표 기반 절대 위치로 배치 (레이아웃 보존)
- `images/` 폴더: 이미지 파일
- `files/` 폴더: PDF/엑셀 등 파일

### 프로젝트 관리
- 다중 프로젝트 (사이드바)
- 삭제 시 인라인 확인 UI ("삭제할까요? [삭제] [취소]")
- 사이드바 상단 돋보기 아이콘으로 검색 패널 열기

### 도움말
- 툴바 `?` 아이콘 클릭 → 기능 설명 모달 팝업
